(tuple)

Tuples in Python store many values inside one variable like lists, but the tuple elements cannot be modified (immutable).

Stuff Python tuple can store:

  • Numbers (int and float)
  • Strings (str)
  • Booleans (bool)
  • Lists (and yeah, also the tuple can)
  • Dictionaries
  • Class instances

Tuple list has some good things...

Since tuple is immutable,

  • It can be used for heterogenous data types, while lists usually used in homogenous data types.
  • it is faster than lists when doing some iterations.
  • it can be used as a key for a dictionary.
  • the elements inside it will be write-protection so it prevents unintended changes.

Some functions can be used in tuple:

  • all() - checks the tuple elements:
    • returns True if the tuple is empty.
    • returns True if all elements in a tuple are True
  • any() -checks the tuple elements:
    • returns False if the tuple is empty.
    • returns True if any elements in a tuple are True
  • enumerate() - return enumerate object containins key and values pair for each iterable element.
  • len() - get the numbers of elements inside tuples.
  • max() - get the maximum element value inside the tuple
  • min() - get the minimum element valus inside the tuple.
  • sorted() - return new sorted list
  • sum() - sum all integer elements.
  • tuple() - convert everything inside brackets into tupe.

How do we define tuples?

# define empty tuple
empty_tuple = ()
print(empty_tuple) # outputs ()

# define a tuple
tuple1 = (1, 3, 2, 5)
print(tuple1) # outputs (1, 3, 2, 5)

# define a heterogenous tuple
heterogenous_tuple = (121, "Bill", ['sauce', 'cabbage'], True)
print(heterogenous_tuple)  # outputs
                           # (121, "Bill", ['sauce', 'cabbage'], True)

list.append(x)

Adds an element x in the end of the list.

Let's go back into fruits list.

fruits = ["apple", "banana", "cherry", "lemon"]

Then add an element using .append().

fruits.append("kiwi")
print(fruits)  # outputs ["apple", "banana", "cherry", "lemon", "kiwi"]

list.clear()

Remove all items from the list.

fruits = ["apple", "banana", "cherry", "lemon"]

fruits.clear()

print(fruits) # outputs []

list.copy()

Return a "shallow copy" of the list.

fruits = ["apple", "banana", "cherry", "lemon"]

my_fruits = fruits.copy()

print(my_fruits)
# outputs
# ["apple", "banana", "cherry", "lemon"]

list.count(x)

Return the number of x times appeared in the list.

fruits = ["apple", "banana", "cherry", "lemon"]

my_fruits = fruits.copy()

print(my_fruits.count("apple")) # outputs 1
                                # bcoz' there is 1 "apple"

list.extend(iterables)

Extend the list by appending all the items from the iterable (lists, dictionaries, sets, etc.).

fruits = ["apple", "banana", "cherry", "lemon"]
another_fruits = ["grape", "papaya", "watermelon"]

# append another_fruits list
## into fruits
fruits.extend(another_fruits)


print(fruits) # outputs ['apple', 'banana', 'cherry', 'lemon',
              #          'grape', 'papaya', 'watermelon']

list.index(x, [start, [end]])

Return zero-based index in the list of the first item whose value is equal to x.

fruits = ["apple", "banana", "cherry", "lemon"]

# find index of "lemon"
print(fruits.index("lemon")) # outputs 3

list.insert(i, x)

Insert an item x at a given position i.

fruits = ["apple", "banana", "cherry", "lemon"]

# insert "blueberry" on index 2
fruits.insert(2, "blueberry")
print(fruits) 

# outputs
# ['apple', 'banana', 'blueberry', 'cherry', 'lemon']

list.pop([i])

Remove the item at the given position i in the list.

fruits = ["apple", "banana", "cherry", "lemon"]

# remove item in index 1
fruits.pop(1)
print(fruits) 

# outputs
# ['apple', 'cherry', 'lemon']

list.remove(x)

Remove the first item from the list whose value is equal to x.

fruits = ["apple", "banana", "cherry", "lemon"]

# remove "apple"
fruits.remove("apple")
print(fruits) 

# outputs
# ['banana', 'cherry', 'lemon']

list.reverse()

Reverse the elements of the list in place.

fruits = ["apple", "banana", "cherry", "lemon"]

# reverse fruits order
fruits.reverse()
print(fruits) 

# outputs
# ['lemon', 'cherry', 'banana', 'apple']

list.sort(key=None, reverse=False)

Sort the items of the list in place.

fruits = ["apple", "cherry", "lemon", "banana"]

# sort fruits order
fruits.sort()
print(fruits) 

# outputs
# ['apple', 'banana', 'cherry', 'lemon']
